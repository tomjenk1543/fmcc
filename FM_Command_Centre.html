Settings & Setup

Admin tasks for configuring the app

Import Save Data

Starting a new save (or setting this up for a friend)? Follow the "New Save Data Collection Guide" to capture what FM needs, then hand the screenshots to Claude in a new chat and ask it to fill in the JSON schema from that guide. Paste the result below — this replaces every dashboard, squad and tactic value in the app at once. The bundled Metalul Buzău data is just the starting example.

Save / Load Full Backup

Downloads everything currently stored locally — the imported save, scouted shortlist, position group overrides, Tactic Builder's tactic, the uploaded crest, and your Attribute Threshold setting — as one JSON file. Load it back in any time (after clearing, importing a different save, or on a different computer) to restore this exact state, without re-capturing screenshots or re-pasting JSON.

Club & League Graphics

Auto-load a club crest or league/competition logo straight from FM's graphics folder by ID — handy if you're using a custom logo pack. You can also just click the crest in the hero card to upload an image directly.

Attribute Threshold

An attribute below this value counts as a squad-wide gap on the Tactics and Squad Breakdown pages — lower it for a weaker squad, raise it if you expect more. One value applies across the whole app.

Data Sources

This app doesn't invent FM26 data — role attributes, formations and role pairings all come from real community guides. Worth a visit for anything deeper than what's shown here.

  • sortitoutsi — FM26: Important Attributes for Every Role Key/Preferred attribute lists behind every role in the Tactics and Tactic Builder pages.
  • fmsidekick — FM26 Formation Database The 12 formation presets (and their real pitch coordinates) offered in Tactic Builder.
  • FM Blog — FM26 Role Synergy Guide The role pairings behind Tactic Builder's Role Synergy panel and Recommend Roles bias.
